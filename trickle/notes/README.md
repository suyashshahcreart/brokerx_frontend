# Elite Property Brokers Website

A premium real estate broker website built with Bootstrap 5, featuring a modern and elegant design.

## Pages

- **Home** (`index.html`) - Hero section, featured properties, search, and key facilities
- **Properties** (`properties.html`) - Property listings with filtering options
- **About** (`profile.html`) - Broker profile, credentials, and client testimonials
- **Service Areas** (`areas.html`) - Interactive map showing service coverage
- **Contact** (`contact.html`) - Contact form and contact information

## Design Features

- Premium color scheme with navy blue primary and gold accent
- Mobile-first responsive design using Bootstrap 5
- Lucide icons for clean, modern iconography
- Interactive map using Leaflet.js
- SEO-optimized with meta tags

## Customization

### Colors
Edit CSS variables in `styles.css`:
- `--primary-color`: Main brand color
- `--secondary-color`: Accent color

### Content
- Replace placeholder images with real property photos
- Update broker profile information in `profile.html`
- Add actual property listings in JavaScript files

### Map
Update coordinates in `areas-app.js` to show your actual service areas and properties.
